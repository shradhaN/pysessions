### Downloading and loading files in jupyter

**Requires Miniconda/Anaconda to be installed**

    $ git clone https://github.com/lfapython/pysessions.git
    $ cd pysessions
    $ git submodule init
    $ git submodule update
    $ conda env create
    $ source activate pysession
    $ jupyter-notebook
